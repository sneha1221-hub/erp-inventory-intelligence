# AI-Powered ERP Inventory Intelligence System

## Overview

An enterprise inventory intelligence platform that allows users to query inventory data using natural language, generate demand forecasts, and receive automated reorder recommendations.

## Features

- Natural language inventory queries
- ERP data integration
- Demand forecasting
- Automated reorder recommendations
- Inventory analytics

## Tech Stack

- Python
- FastAPI
- XGBoost
- NLP
- PostgreSQL
- Streamlit

## Architecture

User Query
↓
NLP Engine
↓
Inventory Database
↓
Forecasting Model
↓
Recommendation Engine
↓
API Response
